1. Knowledge Hub Overview

Knowledge Hub provides divided two areas which are Personal area and Category area. A Personal area provides the best knowledge and recommended knowledge to individual based on accumulated knowledge information in the company. A category area provides categorized information by tags. The best knowledge is provided according to the best date standard management by manager in knowledge monitor. Recommended knowledge is viewed with matched knowledge according to the 'interest' tags which are registered in profile. All matched knowledge with tags to the relevant category are viewed on the category area, it is managed through category management function which is in manager's area.

Knowledge Hub main screen configuration

The following picture is knowledge Hub main screen configuration. Knowledge information can be checked through searching tags. Tag cloud, the best/recommended knowledge are viewed. Click each function and check detailed information.

  • ① Find Knowledge

    Input tags which a user desire to search, knowledge information about the relevant tags is viewed.

  • ② Tag Cloud

    Tags which a user searched for are viewed with cloud.

  • ③ The best knowledge

    View selected the best knowledge. Click each one and check detailed information.

  • ④ The recommended knowledge

    The relevant information is viewed based on interested tags registered in profile by users.

2. Knowledge Hub Menu configuration

Knowledge Hub are consisted of menu for all users and management menu for managers. All users enable to check the best knowledge and the recommended knowledge through personal area. In category area, checking knowledge information by tags is feasible. A category management which is in the Administrator menu is provided only for users with authority.

  • ① Personal

    Consist of The best knowledge and the recommended knowledge.

  • ① Category

    View a category tree registered by manager. Click the relevant category and it moves to the detailed pages.

  • ② Administrator

    A manager provides a function to manage categories in Knowledge Hub.

3. Personal

3.1 The best knowledge

Provide the best knowledge list according to the best data standard management set by knowledge monitor's manager. Each knowledge has a score. Click and detailed information will be provided.

3.2 The recommended knowledge

Provide the recommended knowledge list according to interested tags which are registered in user's profile. The recommended knowledge can be changed and provide the new list.

4. Category

Click each category and users enable to move the relevant pages. Each category owns a tag. Like registered tags which have tag information view lists of knowledge. Click and open detailed information.

5. Administrator

5.1 A category management

A function is provided for manager to manage category area. Click right on the mouse and add, modify and delete categories. It also enables to changed the order of category with Drag&Drop and create a tree structure. Input each name of category and tag, the relevant knowledge will be searched in the category and ranged automatically. Also, select a search object (Q&A, BBS, Café, Forum, etc.) and extract included knowledge in wanted area.
