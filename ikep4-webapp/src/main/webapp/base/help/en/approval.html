1. RSS 개요

RSS는 웹상에서의 뉴스 기사나 소식을 전하는 사이트 또는 개인 블로그와 같은 자주 갱신되는 디지털 컨텐츠를 전파하기 위해서 XML 형태로 기술되는 표준 규약입니다. 컨텐츠 공급자는 이용자들이 갱신된 컨텐츠 정보를 받아갈 수 있도록, RSS 규약에 맞는 컨텐츠 갱신 정보를 웹으로 접근 가능한 URL(Feed URL)을 통해서 제공하고, 이용자들은 RSS 규약을 이해하는 구독기(Reader)를 사용해서 Feed URL을 등록하면 해당 사이트나 페이지에 직접 방문하지 않고도 새로운 정보를 받아볼 수 있습니다.

iKEP 4.0은 외부 RSS Feed를 구독할 수 있는 RSS Reader와 iKEP 4.0 내부의 컨텐츠를 구독할 수 있도록 Feed를 제공해 주는 내부 컨텐츠용 RSS Feed Generator를 제공합니다.

3. RSS Reader

3.1 채널 관리

iKEP 4.0이 제공하는 RSS Reader를 통하여 컨텐츠 갱신 정보를 구독할 수 있습니다. 컨텐츠를 구독하기 위해서는 구독하기 원하는 사이트나 iKEP 내부의 컨텐츠 페이지에서 제공하는 Feed URL을 Reader에 등록하여야 합니다.

컨텐츠를 구독하기 위해 등록하는 Feed URL을 iKEP 4.0 Reader에서는 채널이라고 부르며, 채널정보를 등록해 두면 Reader를 통해서 해당 채널에 새로운 글이 올라오면 그 사이트에 직접 방문하지 않고도 새로운 글을 바로 받아볼 수 있게 됩니다.

iKEP 4.0 RSS Reader 채널 등록 단계

① RSS Reader 채널목록의 'Edit' 화면으로 이동합니다.
② 하단의 채널등록 영역의 URL 입력란에 등록하고자 하는 Feed URL을 붙여넣기하거나 직접 입력합니다.
③ Feed URL 입력 후에 옆의 '주소체크' 버튼을 클릭하여, 입력한 Feed URL이 올바른 URL인지 확인합니다.
④ 채널명 항목에 입력한 Feed URL을 채널 목록에서 표시할 때 사용할 이름으로 입력합니다. 채널명에 입력한 이름으로 채널 목록에 표시됩니다.
⑤ '추가' 버튼을 클릭하여 등록을 완료합니다. 만약 입력한 Feed URL이 '주소체크'를 통과하지 못하면 등록되지 않습니다.
⑥ 만약 입력을 취소하고 새롭게 입력을 시작하고자 한다면 '취소' 버튼을 클릭합니다.
⑦ 등록이 완료 되었으면 원하는 위치로 조정합니다. 채널 순서는 Drag & Drop으로 채널 항목을 움직여서 원하는 위치에 가져다 놓은 방식으로 지정 가능합니다.
⑧ 채널 관리용 'Edit' 화면을 닫고, Reader의 채널 목록에서 새로 등록한 채널명을 확인하고 클릭합니다.
⑨ 'Refresh' 버튼을 클릭하면 가장 최근 등록되거나 변경된 정보를 보여 줍니다.
restriction_icon 제약 사항
  • 개인 사용자 별로 최대 10개의 채널을 등록하여 사용할 수 있습니다.
  • 최대 보관 개수는 사용자의 채널 별 최근 항목순으로 최대 1000개 까지 입니다.
