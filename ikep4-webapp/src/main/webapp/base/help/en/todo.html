1. To-Do Overview

The system supports To-Do function for easier performance of personal to-do management and to-do order management.

2. To-Do menu configuration

① Move to new to-do order screen.
② Move to screen to search to-do list assigned to you, and new To-Do counts will be displayed.
③ Move to screen to search to-do I ordered.

3. To-do registration

This is to give to-do order to yourself or/ and another person and same to-do can be ordered to many users at a time.

The followings are description of each input item based on example screen for question registration.
① title : Enter To-do title. Title is required item.
② to-do deadline : Enter deadline of the to-do. Deadline can be set by minute unit.
③start date : Set to-do start date. To-do start date will be today or any day after today.
④ person in charge : Designate person in charge. One person in charge can be set.
⑤ to-do detail input : Enter to-do detail.
⑥ file attachment : Select and attach file related to to-do.

4. Progress management

This is to check item assigned to you in the list and can enter and complete progress detail in detail to-do list screen.

Checking My To-Do list

You can search To-Do list assigned you. You can check to-do requestor, title, request date, closing date and progress status based on closing date.

Progress registration

Enter progress detail in to-do detail box and click ‘to-do completed’ on the bottom of the screen to complete to-do.
