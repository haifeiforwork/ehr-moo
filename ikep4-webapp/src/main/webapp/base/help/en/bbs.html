1. How to use board

1.1 Basic method to use board

Any user who has permission to write an item in board can register new item in the relevant board to share it with other users.

How to registration new item

① Enter title.
② Select posting period. 3 month period is set as default.
③ Enter tag. Tag is a keyword that represents an item.
④ Enter contents. You can use table, figure registration and other functions of editor.
⑤ If you want to attach a file(s), attach file by dragging it from Explorer.

1.2 How to search item

You can select one of general list type, gallery type, contents type and simple list type that provides only minimum information. You can select one of types according to your taste or purpose.

General list search

① There are 4 types of list search as below.
- General list
- Simple list
- Summary list
- Gallery list
② If you want to view according to my use during list search, select a layout with detail search screen.
- List views
- Lengthwise view
- Widthwise view

1.3 Making use of item

Since data in an item are valuable information and knowledge for your business, the system provides functions to share them as information and knowledge.

Recommending useful data

If you think that item contents are helpful, please click ‘Recommend’ button on the right top of the item. Click ‘Yes’ to the question of ‘Will you recommend item?’ to add ‘1’ to the number of recommendation. Only 1 recommendation is available for 1 item. No more recommendation is available if you have already recommended it.
The above ‘recommendation’ function is provided because information recommended by many users can be useful to a lot of users.

Writing comment

In case a board is register to allow comment, comment input window will e provided on the bottom of an item.
Enter comment and click ‘Save’ button.
If you want to add additional opinion to a comment, please click icon.

1.4 Sharing and spreading of item

You can share and spread item contents by via mail, note, etc.

Spreading item

A favorite icon and icons to send the item to mail, note, micro blog will be displayed on the right top of the item. Select media to spread the contents to display a window in which you can select target, and click ‘Send’ to send item contents.
In case of mail, 2 options are provided. You can send item information as a link or send item contents as a mail.
In case of note and micro blog icon, the are not available if note and micro blog function is not established.

2. Board management function

2.1 Board configuration management

A board administrator can directly compose and manage board suitable for users.

board management tree

① Board management - management menu provided only for board administrator.
② Management tree to manage board in Level structure type.
• Board creation – place mouse cursor on the higher level board name and click ‘board creation’ menu to create new board. To create a board without a higher level board, place mouse cursor on ‘Category’ on the top of management tree and right click to select ‘board creation’ menu.
• Board deletion – place mouse cursor on the board and right click ‘board delete’ menu to delete it.
• Board modification –place mouse cursor on board name and right click the button to select ‘Modify’ menu to modify attribute information of a board.
• Name change – right click the button to select ‘Name change’ to change board name on the management tree.
• Board position change –Select a board on the management tree and then drag it to desired position.

2.2 Board attribute management

This screen is about detailed description of options to newly create or modify a board.

Board registration

• Board name – Enter name
• Board description – Enter description of a board. If you don’t want it , leave it blank.
• Board list type – Select basic search type for users. It is to set default type by an administrator. A user can change list type.
• Anonymity registration – Select whether to allow anonymous registration of an item or not.
• RSS support – Decide support of RSS Feed URL to notify registration of a new item via RSS Feed. In case RSS support is selected, users can sign up for RSS Reader of registration of a new item in the relevant board via RSS 2.0 or Atom 1.0 type.
• Attach file size – Designate max attach file size.
• Image attach file size – In case image is included in item body, designate max file size.
• Width of image file – In case that width of original image is above designated max length, it will be automatically re-sized according to designated max length.
• Items per a page – Set default number of items per a page. A user can the number of items per a page.
• Reply – Select whether reply is allowed or not.
• Comment – Select whether comment is allowed or not. In case this option is not allowed, comment for the relevant item is not allowed.
• Attach file option – Register extension of a file that can be attached step by step or select one in extension group.
• Allowable file format - Set allowable extension for file attach. Please refer to the followings.
1) All (No limit to extension for file attach)
2) txt (text: asc,txt,text,diff,log,htm,html,xhtml,rtf,xml)
3) img (image: bmp,gif,jpeg,jpg,png,tiff,tif)
4) doc (document: doc,pdf,ps,ai,eps,rtf,xls,ppt,pps,pot,docx,pptx,xlsx,hwp)
5) ado (audio: wav,wma,mpga,mpega,mp2,mp3,m4a)
6) vod (movie: avi,mkv,mpeg,mpg,mpe,qt,mov,mp4,m4v,flv,rv,swf,swfl)
7) comp(compressed file: zip,alz,rar,jar)
8) app (executable file : com,exe)
9) all (all files)
• Reading permission – Set users who is allowed to read list and item of the relevant board. In case it is set “Open”, all users logged in the system can search it.
• Writing permission – Set users who is allowed to register item in the relevant board. In case it is set “Open”, all users logged in the system can register new item in the relevant board.
