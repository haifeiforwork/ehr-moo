1. Online poll outline

Simple and convenient Online Poll function is provided for Quick collection of opinion and preference survey.

Everybody can create desired vote for collection of opinion and he/she can close collection of opinion any time. To prevent imprudent collection of opinion, administrator will monitor voting activities and stop unnecessary vote anytime

You can create new vote to induce participation of users. Information on created vote will be provided in the following form, and you can easily check vote progress status or participation status . Information on vote in progress or finished on-line vote will be provided as below. In case that you are not participated in any vote yet, ‘vote’ button will be displayed. In case that you are already participated in vote , ‘vote’ button will be disappeared. To view result the relevant vote, click ‘result view’ button.

2.2 How to write new vote

Everybody can create and register new vote in vote composition screen by clicking ‘vote registration’ button. ‘vote period’ information is required. Compose and register vote and click ‘start’ for participation of other users. Before you start a vote, nobody can participate in real vote. However, even it is after ‘start’, nobody can participate in real vote if it is before vote period

on-line vote composition and registration

① Enter questions for your vote.
② Click Vote period by clicking calendar icon. date selected in the front will be start date and the date in the rear will be end date. If specific period is not selected, 1 month of vote period starting from the current date will be set
③ Enter answers for selection of other users step by step. If you don’t like any of entered answers, select that answer and click ‘Delete’ button to remove it.
④ In case it is for everybody, select ‘public vote’ and in case you want participation of specific group[, please select ‘private vote’
⑤ Please select chart form in which vote result will be displayed.
⑥ If you want to preview a vote composed by yourself, click ‘preview’ button.
⑦ If everything is OK, click' save’ button to save it.
⑧ If you click ‘list’ button, it won’t be saved. If you click ‘list’ button you will move back to the previous screen.

Example of on-line vote composition and preview

2.3 vote implementation and close

If you just compose and register a vote, nobody can participated in the vote. Make sure to click ‘start’ after registration. However, even though it is started, still nobody can participated in the vote before vote start date. Vote will be closed on vote close date and no more participation is not allowed. However, you can close the vote by clicking ‘close’ button even when vote period is not over. ‘start’ and ‘close’ of a vote can be done only by vote writer or administrator who are working on the relevant vote

Starting on-line vote

① After conducting ‘save’ of the vote composed in vote registration screen, please move to ‘My Online Poll’ in which you can check the vote registered by yourself.
② In case of a vote that is registered but not started, ‘vote open’ button will be displayed.
③ In case another user want to participate in vote, click ‘vote open’ button. Now, it starts (Open) allowing participation of the user.
④ The contents of a vote can be modified or deleted any time before the vote starts.

3. Function Summary of each on-line vote user

The followings are summary of functions provided by on-line vote from point of view of vote writer who compose and manage, vote participant.

Voting process

  • Preparing : composed but not started yet. ‘Start’ must be done by writer or administrator and it will be changed to 'In progress' status in which users can participated in the vote
  • Waiting : vote has been started, but it’s not vote start date yet
  • In progress : vote has been started, and it is vote period. writer and administrator can delete or close it. After vote period, it will be automatically closed
  • close : manual close auto close on the expiration date by  the writer or administrator. Voting is not available
