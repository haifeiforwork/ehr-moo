1.2 Mail function

Mail service provides functions such as mail sending / receiving, mail view, mail archive.

1.2.1 Functions

Receive box : It provides functions such as mail list, body view, unread mail view, mail composition, delete, mail archive, and forward.
① Unread mail: Display unread mail list.
② Body view: Display mail body on the bottom of the list.
③ Refresh: refresh mail list to the latest list.
④ Search: Can search mail by title/sender/receiver/reference/body.
⑤ Sort: Click attach file image/sender/title/receiving date and time/size on the top of mail list to sort the list.
⑥ Mail reading: Click mail title to view the relevant mail.
⑦ Multi selection: Can select many mails and delete/archive them using checkbox.
⑧ Delete: delete the selected mail.
⑨ Important mail: Mark the selected mail as ‘important mail (asterisk)’.
⑩ Unread marking: Mark as unread mail and title and in bold font.
⑪ Mail archive: can archive the selected mail in PC or copy/move to another mailbox.
⑫ Forward: forward the selected mail to another person including attach file.
⑬ Mail import: Import email file archived in PC to mailbox in webmail.

Mail reading

① Previous: read previous mail.
② Next: read next mail.
③ Reply: composite a reply for mail you are reading.
④ Reply to all: composite reply for the mail you are reading to all of sender/receiver/referrer.
⑤ Mail archive: Archive mail you are reading in PC or archive it to another mailbox.
⑥ forward: forward mail you are reading to another person including 의 attach file.
⑦ Delete: delete mail you are reading.
⑧ Cancel sending: withdraw sent mail.
⑨ Print: print mail you are reading.
⑩ List: display mail list.
⑪ Remove mail attach: remove attach file out of mail and archive it.
⑫ Add to address book: add the sender of mail you are reading in address book.
⑬ Attach file: Can download or read attached file.
Outbox : Outbox list - Check unsent mail list among reserved mails to be send out.
Outbox : Read temporarily saved mail and modify/send it.
Receipt notification box : Can check whether sent mail is read by receiver.(In case of some outside mails, receipt notification function is not available.)

Mail composition

Mail composition : address book provides functions such as receiver selection, recently sent list, simple address book, reference, blind carbon copy, individual sending, reserved sending, web editor, file attach and temporarily saved mail during mail composition will be saved in Outbox.
①  Send: Click [Send] after composing mail to send the mail.
②  Preview: Can check mail you are writing in web screen.
③  Temp. storage: Temporarily store mail you are writing in [Outbox].
④  Option setting: Set reply request functions of individual sending/reserved sending.
⑤  individual sending: In case plural receivers are designated, function to display receiver only
⑥  reserved sending: function to send mail on designated date and time
⑦  Reply request: request for reply by designated time.
⑧  Expiration date: mail will be automatically deleted after designated expiration date.
⑨  List: Move to screen to view mailbox list.
⑩  LDAP: Can search employees of group companies.
⑪  Important: Will be marked as important mail.
⑫  Receipt notification: Can check whether the receiver has checked the mail.
⑬  Add group: Can add designated receiver in address book as group.
⑭  Organization chart: Can select receiver in organization chart.
⑮  Recent sending: Display receiver list of mail that is recently sent.
⑮  Simple address book: Display receiver list set in simple address book.
 Configuration setting
Ο  List setting: Set number of mails displayed in 1 page in mailbox list.
Ο  Receipt notification: Set whether to set receipt notification function as default or not.
Ο  Attach location setting: Can set location of file attach in mail composition screen.
Ο  Permanent delete: If permanent deletion is set, mail deleted in mailbox won’t move into ‘Deleted Mailbox’ but will be permanently deleted.
Ο  Signature management: register signature and set to use it or not. Registered signature will be automatically inserted when you composite mail.
Ο  Mailbox management: Can check usage of each mailbox, save mailbox in PC or delete archived mail.
Ο  Auto sorting: Can sort received mails according to set rules.
Ο  Auto forward setting: Can forward received mail to designated mail account, approval of administrator is required.
Ο  Folder management: Can add/modify/delete mailbox.
