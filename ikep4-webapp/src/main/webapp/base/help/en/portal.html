1. Portal outline

Portal is space for sharing of personalized communication /information to provide not only person centered job data needed for each of employee but also integrated channel for communication

Personal Portal Service Model

Major Features
  • Widget Configuration: Provide personalized information contents
  • Rich Profile Link: Profile and related update information on personal connections
  • Activity Stream: Provide personal presence and change information

3. Person Service

3.1 My Menu

My menu supports personal search area to check recent visit record and quick link of each user for quick access to information.

① Personal information display area
  • personal picture, name, job position
  • department and time setting information
② My Place
  • personal profile, blog, micro blog link
  • Workspace and Café list
③ My History
  • user list and contents list I visited recently

3.2 Quick Menu

Quick menu supports function to select quick menu needed by the user to increase necessity according to needs of each person.

How to set Quick Menu

① Click setting icon on the right side of Quick Menu.
② Quick Menu desired to be added or deleted on the screen.
③ Adjust Order of the selected Quick Menu by dragging and dropping.
④ Click ‘Save’ button to save the changes.

3.3 Personal Menu

Personal menu provides function to search user and document related to the user in one place by providing document(Personal) registered by the user, user/document(Recent) searched recently, favorite user/document(Favorite)

① Personal: contents list created by the user
  • Document, File, Image, Video: document and attach file created by user
  • Comment: reply registered by the user
  • Feedback: reply list of replies to an article registered by the user
② My History
  • People: list of users visited recently (profile visit, etc)
  • Document: list of recently searched document
③ Favorite: user and document list registered in favorite

3.4 Portlet arrangement

Portlet arrangement supports function to set contents needed in the current screen.

How to arrange portlet

① Select and click a portlet in portlet list.
② selected portlet will be displayed on the left top of the screen.
③ To rearrange the position of portlets, drag and drop portlets.
④ Click ‘Reset’ button to display default portlet setting value set by the system.

3.5 Theme change

Theme desired to be used can be selected in the current screen without moving to separate setting screen.

How to change theme

① Click ‘THEME’ link to move to theme setting screen.
② Click ‘Select’ button to complete theme change.
