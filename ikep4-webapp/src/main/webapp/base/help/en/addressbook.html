1. Use of personal address book

1.1 How to use personal address book

You can register and easily manage address that you frequently use in personal address book. After registering address, you can find address in alphabetical index.

address book outline

① address registration – Click this button to move to new address registration screen.
② Setting personal address book – manage personal address book group.
③ personal address book search – Check my all addresses and address of each group.
④ public address book search – Check public address information composited by administrator.
⑤ Import and Export of personal address book address – Import or export personal address as an excel file or vCard format.

new address registration

① name – Required input item. Please enter name used for address.
② group – Decide a group for new address. In case there is no selectable group, leave it as ‘[unclassified]’.
③ company – Please enter company name.
④ department – Please enter department name.
⑤ job grade – Please enter job position or job grade information. title information can be also entered.
⑥ email – Please enter email address
⑦ cell phone – Please enter cell phone number.
⑧ Office phone –Please enter phone number for day time.
⑨ Memo - Please enter other additional information, if any.
※ Only name information is required input item. In case of other items, just enter any available information. After entering all information, click ‘Save’ button to save entered information.

2. management of group of personal address book

To manage address by group, each user can make needed group. ‘Setting’ menu below ‘address registration’ button provides management function for personal group.

personal setting function for group management for each person

① Setting – Click setting link to move to ‘personal group management’ screen.
② Click ‘Register’ button to create new group.
③ Select group desired to be modified in group displayed in personal group management list and click ‘modify’ button to modify the relevant group information.
④ Select group desired to be deleted in group displayed in personal group management list and click ‘delete’ button to delete the relevant group information.

3. Adding in-house employee in address group

Since in-house employee information is already registered in organization chart, no need to directly register it as a new address. Make required group and select another employee to be included in the relevant group in organization chart to add in-house employee information in address group.

Adding in-house employee information in address group

Select setting > group and modify > personal group modify screen
You can add in-house employee in address information in personal group modify screen. If you want to delete it, click in-house employee desired to be deleted in address information and click ‘Delete’ button. To apply added or deleted contents, click ‘save’ button to save them.
① Address book button – Used to add in-house employee.
② Contacts button –Used to add any of already registered outside address.
