1. 작업관리 개요

개인업무관리 및 업무지시처리를 간편하게 수행할 수 있는 작업관리 기능을 지원합니다.

2. 작업관리 메뉴 구성

① 새로운 작업 지시 화면으로 이동합니다.
② 나에게 할당된 작업 목록 조회화면으로 이동하며, 신규 작업관리건수가 표시됩니다.
③ 내가 지시한 작업 목록 조회화면으로 이동합니다.

3. 작업 등록하기

업무지시를 자기자신 및 타인에게 지시할 수 있으며 같은 업무를 동시에 여러명에게 할 수 있습니다.

질문 등록을 위한 예시 화면을 기준으로 각 입력항목들을 차례로 설명하면 다음과 같습니다.
① 제목 : 작업 제목을 입력합니다. 제목은 반드시 입력해야 하는 사항입니다.
② 작업기한 : 등록하는 작업의 완료기한을 입력합니다. 완료기한은 분단위까지 설정할 수 있습니다.
③작업착수일 : 작업 착수일을 설정합니다. 작업 착수일은 금일 이후부터 가능합니다.
④ 담당자 : 작업을 수행할 담당자를 지정할 수 있습니다. 담당자는 1명이상 설정 가능합니다.
⑤ 작업내용 입력 : 작업의 상세내용을 입력합니다.
⑥ 파일 첨부 : 작업과 관련된 파일이 있다면 선택해서 첨부할 수 있습니다.

4. 진척관리 하기

자신에게 할당된 항목을 목록에서 확인할 수 있으며 상세 작업목록화면에서 진척내용 입력후 완료처리를 할 수 있습니다.

나의 작업 목록 확인하기

자신에게 등록된 작업관리 목록을 조회할 수 있습니다. 작업 요청자, 제목, 요청일, 마감일을 알 수 있으며 마감일 기준으로 한 진척상태까지 확인할 수 있습니다.

진척 등록하기

처리내역의 작업내용란에 진척사항을 입력후 화면하단의 작업완료를 클릭하면 작업이 완료처리됩니다.
