1. Message 개요

쪽지는 구성원들간의 효과적인 의사소통을 지원하는 기능입니다. 월 발송용량(10GB)안에서 간단한 메시지부터 공지사항 전달 등 다양하게 쓰일 수 있는 쪽지는 내용의 중요도에 따라 작성자가 구분 지어 발송할 수 있으며, 수신된 쪽지는 알림창을 통해 실시간으로 확인이 가능합니다. 또한, 사용자가 중요하다고 생각하는 쪽지는 별도로 보관이 가능할 수 있도록 개인보관함(1GB)을 제공하고 있습니다. 월 발송용량 및 개인보관함 용량은 관리자에 의해 설정이 가능합니다.

Message 메인화면 구성

Message 메인화면 구성은 아래 그림과 같습니다. 좌측 메뉴 영역 외 오른쪽 영역에 목록을 보여주며, 클릭 시 Message 세부 내용을 아래 그림과 같이 제공받습니다.

① 미확인 : 읽지 않은 쪽지만 보여줍니다.
② 검색 : 확인하고자 하는 쪽지를 검색하는 기능입니다. 쪽지의 구분(일반, 공지, 정보, 배포), 보낸 사람, 내용에 따른 검색이 가능합니다.
③프레임 변경 : 화면의 구조(상하/좌우)를 변경해 줍니다. '상/좌'에는 현재 저장되어 있는 쪽지 목록을 보여주며, '하/우'에는 해당하는 쪽지를 클릭하였을 때 내용을 보여줍니다.
④ 쪽지 목록 : 해당하는 쪽지함에 들어 있는 쪽지 목록을 보여줍니다. 긴급, read/unRead, 구분, 보낸사람, 내용, 일시, 첨부용량 등이 표시되어 있습니다. 화면에 보여지는 쪽지의 수, 선택된 쪽지들의 삭제/보관 등을 실행할 수 있습니다.
⑤ 쪽지 내용 : 사용자가 선택한 쪽지의 내용을 보여줍니다.

2. Message 메뉴구성

Message 메뉴는 모든 사용자들에게 제공되는 메뉴와 관리자에게만 제공되는 관리용 메뉴로 구성되어져 있습니다. 관리자용 메뉴인 Administration 메뉴 그룹은 관리 권한이 있는 사용자에게만 제공됩니다.

① 쪽지작성 : 사용자가 새로운 쪽지를 작성한 후, 발송할 수 있는 기능을 제공합니다.
② 받은쪽지함(수신된 쪽지 목록) : 받은 쪽지 목록을 보여줍니다. 설정된 조건에 따라(보여지는 쪽지의 수, 긴급/unRead 체크여부), 혹은 검색되어진 쪽지 목록을 제공합니다. 관리자가 지정한 기간이 지난 수신 쪽지는 수신함에서 자동 삭제됩니다. 기한이 지나도 삭제되길 원하지 않는다면, 보관함으로 옮겨야 합니다.
③보낸쪽지함(발신된 쪽지 목록) : 사용자가 보낸 쪽지 목록을 보여줍니다. 보낸 쪽지의 수신확인 여부도 확인할 수 있으며, 받은쪽지함과 마찬가지로 설정된 조건에 따라(보여지는 쪽지의 수, 긴급/unRead 체크여부), 혹은 검색되어진 쪽지 목록을 제공합니다. 관리자가 지정한 기간이 지난 발신 쪽지는 발신함에서 자동 삭제됩니다. 기한이 지나도 삭제되길 원하지 않는다면, 보관함으로 옮겨야 합니다.
④ 개인보관함(보관된 쪽지 목록) : 발신 혹은 수신된 쪽지 중 보관되어진 쪽지 목록을 보여줍니다. 보관함에 보관 중인 쪽지는 사용자가 삭제하기 전에는 삭제되지 않습니다.

3. 작성 / 발송

좌측 메뉴에서 쪽지 작성을 클릭하면, 쪽지를 작성하고 발송할 수 있는 팝업창이 생성됩니다. 사용자가 내용을 입력하고, 필요에 따라 파일을 첨부하여 간단하게 쪽지를 보낼 수 있습니다.

쪽지 작성을 위한 예시 화면을 기준으로 각 입력항목들을 차례로 설명하면 다음과 같습니다.

① 긴급 여부 : 긴급하게 확인을 요하는 쪽지에는, 보내는 쪽지의 '긴급'여부를 체크합니다. 수신자에게 '!'표시로 나타나게 됩니다.
② 구분 : 쪽지의 성격에 따라, '일반/공지/정보/배포' 중 선택하여 발송합니다. 기본으로 '일반'으로 설정되어 있습니다.
③수신자 선택 : 발송할 쪽지의 수신자를 선택합니다. 수신자 추가 방법은 옆의 'Address' 아이콘을 클릭하여 조직도 화면을 호출해서 조직도 상에서 답변할 사용자를 선택하거나, 바로 사용자 검색을 통해 입력 가능합니다(최대 500명). 선택한 사용자를 취소하고 싶으면, 선택 박스에서 선택한 후 다시 옆의 'Delete' 버튼을 클릭합니다.
④ 내용 입력 : 쪽지의 내용을 입력합니다.
⑤ 파일 첨부 : 수신자에게 보내는 파일이 있다면 선택하여 첨부할 수 있습니다.

4. 수신 알림

사용자가 받은 쪽지는 브라우저 우측 하단에 실시간으로 알림창을 통해 알려주고 있습니다. 알림창을 클릭하여 쪽지의 내용 확인 및 답장이 가능합니다.

5. 활용하기

수신된 쪽지는 보관되는 보관함에 따라 다양한 기능을 가지고 있습니다. 메일, 답장, 전달, 보관, 삭제, 회수 등의 기능을 통해 사용자는 쪽지를 관리할 수 있습니다.

메일보내기

해당 쪽지를 메일에 첨부하여 보낼 수 있습니다. 아래 그림과 같이 메일 작성 시 하단에 자동으로 첨부됩니다.

답장보내기

보낸 사람에게 답장을 보낼 수 있는 기능입니다. 아래 그림에서 처럼 'Original Message'라는 문구와 함께 쪽지 내용이 자동으로 등록됩니다.

전달하기

수신된 쪽지를 다른 사람에게 전달하고자 할 때 사용되는 기능입니다.

보관하기

받은쪽지함에 있는 쪽지를 개인보관함으로 이동합니다. 관리자가 설정한 기한에 따라 수신된 쪽지는 자동으로 삭제되기 때문에 중요한 쪽지는 보관함에 저장하여야 합니다.

회수하기

보낸 쪽지를 회수하는 기능입니다. 상대방이 읽은 쪽지는 회수가 되지 않으며, 이는 해당 쪽지 '수신확인'을 통해 확인 가능합니다.
