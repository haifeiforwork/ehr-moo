1. Profile 개요

각각의 구성원의 기본정보를 효과적으로 공유하고 편리하게 Connect할 수 있도록 연결 정보를 제공하는 공간으로, 각 사용자는 본인의 정보를 쉽게 업데이트 할 수 있고, 방문하는 타 사용자들은 공유하고자 하는 개인 정보와 자료들로 효과적으로 안내되어질 수 있습니다.

Profile의 화면 예시

Profile 화면 이미지에 대한 설명
개인정보영역 : 프로필 사진, 연락처, 메일주소, Location, Locale(언어선택), Timezone, 외부 SNS 계정
활동정보영역 : Current Work, My Specialization, My Career, My Documents
방명록 :Guest Book

2. Profile 이용방법

2.1 개인정보 관리

Profile의 개인정보 영역에서는 프로필 사진, 연락처, 메일주소, Location, Locale(언어선택), Timezone, 외부 SNS 계정 등을 등록/수정 할 수 있습니다.

Profile 개인정보 관리

Profile 개인정보 관리화면 이미지에 대한 설명
편집모드
  • ② 소개글 관리
  • ③ Profile 사진 관리
  • ④ 개인정보 관리
① 소개글 관리 : 간략한 자기 소개글 등록
② Profile 관리
③ 개인정보 관리 : 연락처, Locale, Tiemzone 등 개인 정보 관리

2.2 Profile 화면 Component

Profile 화면 Component 구성

Profile 화면 Component 구성 이미지에 대한 설명
My Documents : 사용자가 등록한 모든 문서
My Images : 사용자가 등록한 모든 이미지

My Followings : 사용자가 Following 하는 사용자(Micro Blogging)
My Followers : 사용자를 Following 하는 사용자(Micro Blogging)
My Fellow Experts : 전문가 태그가 유사한 사용자(Tagging)

My Files : 사용자가 등록한 모든 파일
