1. Portal 개요

임직원 개인이 필요한 개인 중심의 업무 자료는 물론 커뮤니케이션을 위한 통합 채널을 제공하는 개인화된 의사소통/정보공유 공간입니다.

Personal Portal Service Model

주요특징
  • Widget Configuration : 개인화된 정보 콘텐트 제공
  • Rich Profile Link : Profile 및 관련 인맥 업데이트 정보 제공
  • Activity Stream : 개인의 Presence 및 변경 정보 제공

2. Public Service

전체 사용자들이 공통으로 제공하는 포틀릿 영역과 사용자의 권한에 따른 포탈 메뉴를 제공하고 있습니다.

Public Serivce 제공 목록

서비스 제공 상세 내역
① 포탈 메뉴
  • 사용자 권한에 따른 메뉴 제공
② 퀵 메뉴
  • 사용자 권한에 따른 차별화된 퀵메뉴목록 제공
③ 공용 포틀릿 영역
  • 명언 포틀릿
  • 전사 게시판 포틀릿
  • 혁신 이미지 포틀릿

3. Person Service

3.1 My Menu

빠른 정보 접근을 위해 최근 방문 기록 및 개인별 퀵링크를 확인 할 수 있는 개인 탐색 영역을 지원합니다.

① 개인 정보 표시 영역
  • 개인 사진 및 이름 직급
  • 소속 부서 및 시간 설정 정보
② My Place
  • 개인 프로파일, 메모, SMS 링크
  • 가입한 Workspace 및 Café 목록
③ My History
  • 최근 방문한 사용자 목록 및 컨텐츠 목록 제공

3.2 Quick Menu

필요한 퀵메뉴를 선택할 수 있도록 지원하여 개인별 필요에 따른 필요성의 강화합니다.

Quick Menu 설정 방법

① Quick Menu 우측의 설정 아이콘을 클릭합니다.
② 화면에 추가하거나 삭제할 Quick Menu를 선택합니다.
③ 선택된 Quick Menu의 순서를 드래그 & 드랍으로 조정합니다.
④ 변경된 설정을 저장하기 위해서는 저장 버튼을 클릭하여 반영합니다.

3.3 Personal Menu

사용자가 등록한 문서(Personal), 최근 조회한 사용자/문서(My History), 즐겨찾기한 사용자/문서(Favorite)을 제공하여 자신과 관련된 사용자 및 문서를 한곳에서 조회할 수 있는 기능을 제공합니다.

① 사용자가 생성한 컨텐츠 목록 : 사용자가 생성한 컨텐츠 목록
  • Document, File, Image : 사용자가 생성한 문서 및 첨부파일
  • Comment : 사용자가 등록한 댓글
② My History
  • People : 최근 방문한 사용자 목록 (프로파일 방문 등)
  • Document : 최근 조회한 문서 목록
③ Favorite : 즐겨찾기에 등록한 사용자 및 문서 목록

3.4 포틀릿 배치

현재 이용하고 있는 화면에서 필요한 콘텐츠를 바로 설정할 수 있게 지원합니다.

포틀릿 배치 방법

① 포틀릿 목록에서 선택할 포틀릿을 클릭합니다.
② 선택된 포틀릿은 왼쪽 화면 상단에 나타납니다.
③ 포틀릿의 위치를 변경할려면 드래그 & 드랍으로 위치조정을 합니다.
④ Reset 버튼을 클릭하면 시스템에서 설정한 초기 포틀릿 설정값으로 설정합니다.

3.5 테마 변경

별도의 설정 화면으로 이동 없이 현재 이용하고 있는 화면에서 사용하고자 하는 테마를 바로 선택할 수 있습니다.

테마 변경 방법

① THEME 링크를 클릭하여 테마설정 화면으로 이동합니다.
② 선택 버튼을 클릭하면 테마 변경이 완료됩니다.
