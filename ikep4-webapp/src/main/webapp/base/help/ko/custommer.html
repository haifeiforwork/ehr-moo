1. 고객정보관리

1.1 개요

고객정보관리는 인물정보,주요판매처,고객별 상담내역에서 사용되는 주요 고객의 정보를 관리합니다. 고객의 기본정보를 조회, 신규저장, 수정, 삭제 할 수 있고 SAP 연동을 통하여 SAP에 있는 고객 정보를 동기화 할 수 있습니다. 정보의 수정, 삭제는 관리자와 글 등록자만 가능합니다.

고객정보 조회

① 인물정보 : 해당 고객사로 등록된 인물정보 건수를 보여줍니다. 클릭 시 인물정보 리스트를 볼 수 있습니다.
② 주요판매처 : 해당 고객사로 등록된 주요판매처 건수를 보여줍니다. 클릭 시 해당 고객사 주요 판매처를 조회 할 수 있습니다.
③ 상담이력 : 해당 고객사와 관련된 상담이력 건수를 보여줍니다. 클릭 시 해당 고객의 상담내역을 조회 할 수 있습니다.

1.2 고객정보 등록

고객정보 등록 프로세스는 하나의 고객이 여러 개의 게시물로 중복 저장되지 않도록 사용자에게 지금 등록 할 고객사가 기존에 등록된 고객사인지 다시 한번 체크 할 것을 권유합니다.

기 등록된 고객인지 확인하기

① 현재 등록하고자 하는 고객이 기존에 등록되었는지 검색을 통해 확인합니다.
② 등록 : 검색 후 등록된 데이터가 없을 때 등록 버튼을 클릭합니다.
③ 취소 : 검색 후 등록된 데이터가 검색 될 때 취소 버튼을 클릭합니다. 자신이 등록하고자 하는 고객정보가 저장되어있으므로 추가 저장은 할 수 없고 조회는 가능합니다.

고객정보 등록하기

① TAB을 클릭하면 각각의 정보를 저장할 수 있습니다. 기본정보를 저장한 뒤 TAB을 이동해야 합니다. 기본정보를 저장하지 않고 TAB을 이동하면 기존에 입력했던 데이터는 모두 사라집니다.
② TAB에 있는 정보를 입력 한 뒤 저장 버튼을 클릭해야 정보가 저장됩니다.
③ 추가 버튼을 누르면 추가적으로 입력할 수 있는 칸이 생성됩니다. 새로 생성된 칸에 아무 정보도 입력하지 않고 저장버튼을 누르면 해당 빈칸은 저장되지 않습니다.

SAP 정보 연동하기

① 기본정보를 저장할 때 SAP에 해당 고객사 정보가 있는지 확인합니다.
② 고객사 정보의 SAP 동기화 유무를 확인 할 수 있습니다. SAP ID가 정상적으로 출력된다면, 해당 고객사는 SAP와 동기화 된것입니다.
③ 등록 할 고객 정보가 SAP에 있는지 검색합니다.
④ 등록 할 고객명을 클릭하면 SAP와 동기화 되고 해당 정보를 가져옵니다.
⑤ 등록버튼을 꼭 클릭하여 동기화된 정보를 저장합니다.

2. 인물정보

2.1 조회 및 등록

인물정보는 고객정보관리에 등록되어있는 고객사의 인물정보를 관리합니다. 고객정보관리에 등록되지 않은 고객사의 인물은 등록할 수 없습니다. 따라서, 등록 전에 고객정보관리에서 해당 고객사를 먼저 등록한 뒤 인물정보를 등록합니다.

인물정보 조회

인물정보 등록

고객명을 입력하고 클릭하면 고객정보관리에 등록된 고객이 검색됩니다. 고객정보관리에 등록된 고객만 인물정보를 등록할 수 있으며 그 외의 회사 인물은 고객정보관리에서 고객 등록 후 등록 할 수 있습니다.

3. 주요 판매처

3.1 조회 및 등록, 수정/삭제

주요판매처는 고객정보관리에 등록된 고객사가 거래하는 판매처 정보를 관리합니다.

주요판매처 조회

해당 고객사의 주요판매처가 몇 건 등록되어있는지 확인 할 수 있습니다.

주요판매처 등록

주요판매처 등록은 최초로 고객사의 주요판매처를 등록한 등록자가 추가 등록할 수 있습니다.
최초 등록 : 고객명은 검색을 통하여 고객정보관리에 등록된 고객사만 등록 가능합니다.
추가 등록 : 기존에 해당 고객사에 대한 판매처를 등록한 등록자, 시스템 관리자가 새로운 판매처를 추가 등록할 수 있습니다.

주요판매처 수정/삭제

① 등록자 및 관리자가 각 판매처의 이름을 클릭하면 수정화면이 생성됩니다.
② 수정화면에서 삭제 버튼을 클릭하면 해당 판매처는 삭제됩니다.

4. 고객별 상담내역

4.1 조회 및 등록

고객별 상담내역은 고객정보관리에 등록되어있는 고객사의 상담내역을 관리합니다. 고객정보관리에 등록되지 않은 고객사의 상담내역은 등록할 수 없습니다. 따라서, 등록 전에 고객정보관리에서 해당 고객사를 먼저 등록한 뒤 상담내역을 등록합니다.

고객별 상담내역 조회

고객별 상담내역 등록

고객명을 입력하고 클릭하면 고객정보관리에 등록된 고객이 검색됩니다. 고객정보관리에 등록된 고객만 상담내역을 등록할 수 있으며 그 외의 상담내역은 고객정보관리에서 고객 등록 후 등록 할 수 있습니다.

5. 품질/클레임 상담내역

5.1 조회 및 등록

품질/클레임 상담내역은 고객정보관리와는 별개로 모든 고객사들의 품질/클레임 상담 내용을 관리합니다.
