1. 전자결재 개요

전자결재는 오프라인에서 paper work으로 진행되던 결재 업무를 웹 환경에서 결재 상신부터 최종 결재 까지 수행 할 수 있도록 지원합니다. 전자결재는 기존의 기간 시스템에서 수행되던 시스템별 전자결재를 통합(통합결재)하여 결재업무의 편의성을 극대화 하고 표준 결재 업무 프로세스를 최대한 통일함으로써 사용자에게 일관된 UI와 결재기능을 제공하게 됩니다.

전자결재 지원 개요

전자결재 서비스 제공 개요

전자결재 서비스 주요특징

  • 상신함/미결함 등 다양한 상태의 문서함을 제공하여 결재 문서관리를 편리하게 할수 있도록 지원합니다.
  • 기존 순차적인 결재 이외에 직/병렬 협의선을 설정하여 의사결정과정에서의 다양한 의견수렴을 가능하도록 지원합니다.
  • 휴가/출장 등 공석 시 위임 기능을 제공하여 업무에 차질이 없도록 지원합니다.
  • 각 결재문서 양식별 설정을 통해서 기본 담당자 지정 기능 등 다양한 기능을 사용할 수 있습니다.

2. 전자결재 기능 범위

전자결재는 품의서 작성, 품의서 결재, 결재위임, 기간위임, 협의지정, 사용자정의결재선 지정등의 기능을 제공하며, ESS 및 CAS 시스템 일부 업무의 통합 결재 기능을 제공 합니다.

3. 전자결재 주요기능

3.1 품의서 작성

전자결재는 업무별 결재 양식을 선택하여 품의 합니다. Each set of forms will be implemented through the forms management.(Admin)

전자결재 품의서 양식 선택

전자결재 품의서 작성

전자결재 품의서 상신 – 전자결재 비밀번호 입력

전자결재 품의서 상신 – 결재요청의견 및 상신

① 양식 유형을 선택합니다. 업무 구분 입니다.
② 양식 구분을 선택합니다. 기안양식/신청양식/품의양식 등이 있습니다.
③ 결재 양식 명으로 검색 할 수 있습니다
④ 양식 명을 클릭하여 해당 양식 문서를 엽니다.
⑤ 설문 결과 공개여부를 선택합니다. 비공개 선택 시, 설문 결과를 사용자에게 제공하지 않습니다.
⑥ 품의서 작성 후 임시저장 할 수 있습니다.
⑦ 결재한 결재선을 설정 합니다.
⑧ 품의서 작성 후 최종 결재를 상신 합니다.
⑨ 해당 결재 양식의 기본 담당자가 출력됩니다. 양식 관리에서 설정 됩니다.
⑩ 협의선 지정시 협의선의 Type을 선택합니다. 병렬 협의 또는 직렬 협의를 선택합니다.
⑪ 결재가 완료된 후 최종 문서를 통보받을 사람이나 그룹을 선택합니다. 양식관리에서 기본 통보자를 선택해 놓으면 여기에 출력됩니다
⑫ 기결재 된 문서를 참조할 수 있게 합니다.
⑬ 기안에 필요한 양식이 출력됩니다. 양식은 양식관리에서 설정 되면 사용자가 변경해서 결재를 상신 할 수 있습니다
⑭ 첨부파일을 선택할 수 있습니다
⑮ 결재상신 시 전자결재 비밀번호를 입력합니다.
(16) 전자결재비밀번호 입력 후 상신 의견을 입력합니다.
(17) 결재 요청 버튼을 클릭하여 최종 상신 합니다.

3.2 문서함

전자결재는 상신함 / 임시보관함 / 미결함 / 기결함 / 위임문서함 / 부서기결함 등의 용도에 따른 다양한 문서함을 제공합니다.

상신함

임시보관함

결함

기결함

위임문서함

부서기결함

① 기안문서 – 상신함 : 본인이 상신한 문서의 목록입니다.
② 문서 검색 조건을 설정하는 영역입니다
③ 문서의 결재 진행 상태를 색상 별 아이콘으로 표시 합니다.
④ 기안문서 – 임시보관함 : 기안 문서를 상신전에 임시 저장한 목록입니다. 삭제할 수 있고 수정 후 결재를 상신 할 수 있습니다.
⑤ 결재문서 – 미결함 : 본인이 결재할 문서의 목록입니다. 제목을 클릭하여 기안문서를 연 후 결재합니다.
⑥ 결재문서 – 기결함 : 본인이 결재한 문서의 목록입니다. 제목을 클릭하여 결재문서 내용을 확인 합니다.
⑦ 참조문서 – 위임문서함 : 본인이 기간위임 또는 문서위임을 통해 위임한 문서함 입니다. 위임기간이 끝나거나 위임을 철회할 경우 미결된 문서는 미결함으로 돌아오며, 수임자에 의해 기결된 문서는 그대로 위임 문서함에 남습니다.
⑧ 참조문서 – 부서기결함 : 본인의 부서에서 상신한 문서 비밀문서가 아닌 일반문서가 결재가 완료된 경우 부서 기결함을 통해 조회 할 수 있습니다. 타 부서 기결함 조회 권한을 부여 받은 경우 다 부서의 기결 문서도 조회 할 수 있습니다.
⑨ 본인이 조회 권한이 있는 부서가 여러 개 일때, 즉 겸직이거나 부서 기결함 조회 권한을 받은 부서가 있는 경우 해당 부서를 선택하여 조회합니다.

3.3 문서 결재

전자결재는 결재 기능을 제공합니다. 본인이 직접 승인/반려 할 수 있고 해당 결재문서를 타인에게 결재를 위임할 수도 있습니다. 결재 시 결재 비밀번호를 통하여 결재문서에 대한 보안을 구현 합니다. 협의자는 단순 검토 및 의견제출을 할 수 있으나 해당 결재문서의 양식설정에 따라 협의반려를 수행 할 수도 있습니다.

품의/기안문서 결재

품의/기안문서 결재 – 전자결재비밀번호 입력

품의/기안문서 결재 – 결재의견 입력 및 승인/반려

품의/기안문서 결재 – 협의문서 협의의견 입력 및 승인/재고려

① 미결함에서 미결문서를 클릭하여 결재문서 내용을 확인하고 결재 버튼을 클릭하여 결재를 수행합니다.
② 현 문서의 결재 권한을 타인에게 위임합니다. 위임 후 미결함에서 현 문서는 사라지며 위임 문서함에서 확인 할 수 있습니다.
③ 결재 버튼을 클릭하면 전자결재 비밀번호를 입력하는 창이 출력됩니다. 전자결재 비밀번호가 설정되어 있지 않은 경우 결재문서로 들어올 때와 결재 버튼을 클릭할 때 결재 비밀번호를 설정하라는 경고창이 출력됩니다. 결재비밀번호는 전자결재 환경 설정에서 설정 할 수 있습니다.
④ 전자결재 비밀번호를 입력 하면 결재의견 입력과 승인 및 반려를 할 수 있는 창이 출력됩니다.
⑤ 결재의견 입력후 승인/반려 버튼을 클릭하여 결재 합니다
⑥ 본인이 협의 결재선인 결재 버튼 대신 합의 버튼이 출력되며, 합의 버튼을 클릭하면 전자결재 비밀번호 입력 후 합의 의견 및 협의/재고려 처리를 할 수 있는 창이 출력된다.
⑦ 협의 의견을 입력 후 협의 또는 재고려 버튼을 클릭하여 결재 합니다. 재고려는 반려 기능입니다.

3.4 환경설정

전자결재는 기간 위임 및 결재 비밀번호 설정을 위한 환경설정 페이지를 제공합니다. 본인이 공석일 때 위임기간과 수임자를 지정하여 결재권한을 위임함으로써 업무에 차질이 없도록 지원하며, 결재비밀번호 설정을 통해 결재 문서의 보안을 구현하였습니다.

전자결재 결재비밀번호 등록

전자결재 결재비밀번호 변경

전자결재 결재비밀번호 초기화

① 위임설정 여부를 선택합니다.
② 부재중 위임 기간을 설정합니다.
③ 수임자를 지정합니다.
④ 위임설정내용을 최종 저장합니다. 위임이 설정이 되면 위임자에게 결재문서가 도달할 시 결재문서는 자동으로 수임자의 미결함에 출력되고, 위임자는 위임한 결재문서함에 출력됩니다. 수임자가 결재 하기 전 위임기간이 끝나거나 위임이 해제 되면 원래대로 위임자의 미결함에 출력되고 수임자의 미결함에서는 사라집니다. 수임자가 기결한 건은 수임자의 기결함과 위임자의 위임한 결재문서함에 그대로 남습니다.
⑤ 전자결재 비밀번호를 최초로 등록합니다. 영문과 숫자를 혼용하여 8자리입니다
⑥ 전자결재 비밀번호를 변경합니다. 기존 비밀번호를 알아야 변경 할 수 있으며, 기존 비밀번호를 모를때는 초기화 버튼을 통하여 비밀번호를 변경하셔야 합니다.
⑦ 초기화 버튼을 클릭하면 본인의 등록된 이동전화로 인증 비밀번호를 전송할 것인지 묻는 창이 출력됩니다.
⑧ 인증 비밀번호 전송 확인 창에서 확인 버튼을 클릭하면 해당 번호로 인증 번호가 전송됩니다.
⑨ 이동전화로 받은 인증번호를 입력하고 새로 사용할 비밀번호를 등록합니다.
