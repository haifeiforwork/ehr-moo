1. Q&A 개요

묻고답하기는 조직의 구성원들이 자율적으로 필요한 사항을 문의하고 답변하는 문제 해결 활동을 지원하는 기능입니다. 보다 신속하고 효과적으로 질문을 공유하고 답변을 얻을 수 있도록 쪽지, SMS, 메일, 마이크로블로그(사내트위터)와 같은 여러 매체를 통해 질문과 답변 내용을 전파할 수 있습니다. 또한 다른 사용자들이 유사한 문제에 대해서 이전에 질문했던 다른 질문이나 관련된 답변이 있다면 이를 함께 제공하여 단순히 하나의 질문과 답변으로 끝나는 것이 아니라 유사하거나 반복적으로 발생하는 질문과 문제들에 대한 종합적인 문제 해결의 장이 될 수 있도록 하였습니다.

Q&A 진행 단계

Q&A에서는 문제해결을 위해 질문을 등록하게 되면, 여러 사람들이 답변을 제시하게 됩니다. 아직 최초 답변이 이루어지지 않았다면, 해당 질문 건은 '미답변' 상태가 됩니다. 제시된 여러 답변 중에서 질문자는 제일 도움이 되었던 답변을 '채택'할 수 있는데, 이렇게 되면 해당 질문 건은 '완료'가 된 것으로 간주됩니다. 질문에 대한 답변은 이루어 졌지만, 질문자가 만족할만한 답변이 제시되지 못해서 질문자가 아직 가장 도움이 되었던 답변을 '채택'하지 않았다면 해당 질문 건은 여전히 '미해결' 상태인 질문이 됩니다.

Q&A 메인화면 구성

Q&A의 메인화면 구성은 아래 그림과 같습니다. Q&A에 접근하게 되면, 좌측 메뉴 영역 외 오른쪽 영역에 Q&A 상에서 이루어지는 최근 활동 내용을 아래 그림과 같이 제공받습니다.

① 검색하기 : 제목, 태그 또는 질문자 항목을 선택한 뒤에 입력한 검색어가 해당 항목에 포함되어 있는 질문 목록을 제공합니다. 검색하기 기능을 사용하면 본인이 궁금하거나 질문하고자 하는 사항과 유사한 내용을 이미 다른 사람이 질문을 해서 그에 적합한 답변을 얻었는지 알아 볼 수 있습니다.
② 질문하기 : 새로운 질문은 등록하려면 '질문하기' 버튼을 클릭합니다. 새로운 질문을 작성할 수 있는 질문 등록 화면으로 연결됩니다.
③ 최근 등록된 긴급질문 : 최근에 등록된 질문 중에서 긴급질문으로 등록된 질문 10건을 선택한 후 롤링하면서 디스플레이 해주는 영역입니다. 긴급질문은 답변이 신속하게 이루어져야 하는 건이므로, 사용자들에게 잘 보일 수 있는 화면 중심부에 표시되도록 하였습니다.
④ 평균답변시간 정보(다른 사용자들에 많이 참조되는 인기있는 질문 목록) : 답변이 이루어진 질문에 대해서 제일 첫번째 답변을 받기까지 평균 시간 정보를 제공하여, 질문을 하게되면 평균적으로 최초 답변을 언제까지 받을 수 있는지 예상할 수 있게 도와줍니다.
⑤ Best Q&A : Best Q&A로 선택된 질문과 답변 정보를 제공합니다. Best Q&A 정보를 제공하는 이유는 업무에 도움이 되는 최근 질문과 답변 활동을 가능한 많은 사용자들과 공유하기 위해서 입니다. Best Q&A는 관리자가 설정한 선정 정책에 의해 선택되는데 이렇게 선택된 질문 중에서 최근에 등록된 순으로 5개 질문을 선택한 후 이 중에서 임의로 하나의 질문을 선택해서 보여줍니다. 그리고, 선택된 질문에 대한 답변이 여러개 있다면 이 중에서 가장 답변에 대한 평가 점수가 높은 답변을 해당 질문과 함께 제공합니다.
⑥최근 등록된 질문 : 최근에 어떤 질문들이 올라왔는지 확인 할 수 있도록 최근 등록된 질문을 5개까지 보여주는 영역입니다. 각 질문앞에 붙어있는 '미답변', '답변중','해결'이라는 말머리표는 질문 상태를 알려주기 위함입니다.
⑦질문 태그 : 최근에 질문 등록 시 사용된 태그 정보를 제공합니다. 최근에 사용된 태그를 최대 20개까지 추출하여 태그 클라우드 속성을 반영하여 디스플레이 하게 됩니다. 태그 클라우스 속성을 반영해서 태그를 디스플레이하게 되면, 최근에 사용된 태그이지만 이미 이전에 다른 사용자들이 많이 사용했던 태그라면 강조되어 디스플레이 됩니다. 다만, 디스플레이 되는 인터페이스는 롤링 되는 슬라이드형 인터페이스로 한 번에 5개씩 보여집니다.

2. Q&A 메뉴구성

Q&A 메뉴는 모든 사용자들에게 제공되는 메뉴와 관리자에게만 제공되는 관리용 메뉴로 구성되어져 있습니다. 관리자용 메뉴인 Administration 메뉴 그룹은 관리 권한이 있는 사용자에게만 제공됩니다.

① 나의 질문 / 나의 답변(개인 Q&A 활동 정보 파악을 위한 본인의 질문과 답변 목록) : 본인이 등록한 질문 개수와 답변한 개수 정보를 제공하고, 클릭하면 각각 본인이 질문한 목록과 답변을 제시한 질문 목록 화면으로 연결됩니다.
② 긴급 질문(질문 등록 시 긴급 질문으로 체크한 질문 목록) : 질문 등록 시에 질문자에 의해서 긴급 질문으로 체크되었던 질문만 모아서 제공합니다. 긴급 질문은 최근에 등록된 질문 순서로 보여집니다.
③ 미해결 질문(미답변된 건이나 아직 채택 단계 이전의 질문 목록) : 질문이 등록은 되었으나 질문자가 만족할만한 답변이 아직 제시가 되지 않아서, 질문자가 아직 완료 처리하지 않은 질문만을 모아서 제공합니다. 질문자는 다른 사용자들에 의해 제시되는 답변들 중에서 가장 도움을 많이 받은 답변을 우수 답변으로 '채택' 하게 되고, 이렇게 질문자에 의해서 우수 답변이 '채택' 되면 해당 질문은 해결된 것으로 간주합니다. 반대로 '채택'이 이루어지지 않은 질문 건들을 미해결된 건으로 간주합니다.
④ Category(분류체계에의한 탐색을 위한 분류별 목록 ) : 분류별로 해당 분류에 해당하는 질문들만을 모아서 조회할 수 있습니다.
⑤ Administration(관리기능 ) : Q&A 관리자에게만 제공되는 관리 메뉴 입니다. 관리 메뉴에는 Category 분류체계를 설정할 수 있는 기능과, 인기 Q&A 및 우수 답변을 선정하는 기준을 관리할 수 있는 정책 관리 기능이 있습니다.

3. 질문하기

편리한 질문 작성을 위해 웹 편집기를 내장하였으며, 그 외에도 질문 작성을 위한 입력화면은 질문한 내용에 대해서 답변이 올라왔을 때 질문자가 바로 확인할 수 있는 알림 설정, 특정 인물을 답변자로 지정할 수 있는 전문가 답변 요청등의 질문과 답변 활동의 편의를 위한 여러 옵션을 포함하고 있습니다.

질문 등록을 위한 예시 화면을 기준으로 각 입력항목들을 차례로 설명하면 다음과 같습니다.

① 제목 : 질문 제목을 입력합니다. 질문 제목은 반드시 입력해야 하는 사항입니다.
② 긴급 질문 : 긴급하게 답변을 요하는 건이라면, 긴급 질문 항목을 체크해 둡니다. 긴급 질문으로 체크된 질문은 긴급 질문 만을 모아서 조회될 수도 있으며, 많은 사람들이 질문을 확인할 수 있도록 최근 질문 건이면 메인화면에 노출됩니다.
③ 분류 선택 : 긴급 질문 체크 항목 옆의 드롭다운 박스는 질문하려는 사항이 어느 분야에 해당하는지 선택하는 부분입니다. 특별히 해당하는 분야가 없다면 '미분류 및 기타'로 선택하면 됩니다.
④ 답변 등록 알람 : 질문자에게 답변이 등록되었을 때 알려주는 기능입니다. 이 알람 기능을 통해 질문자는 답변이 등록되었을 때 가능한 신속하게 답변을 확인할 수 있습니다. 답변이 등록되었을 때 알림을 받을 수 있는 매체에는 메일, SMS, 쪽지가 있으며 이들 중에서 중복으로 매체를 선택할 수도 있습니다.
⑤ 질문 내용 입력 : 질문의 내용을 입력합니다. 웹 편집기가 내장되어 있으므로, 질문 내용을 풍부하게 편집할 수 있습니다.
⑥ 파일 첨부 : 질문과 관련된 파일이 있다면 선택해서 첨부할 수 있습니다.
⑦ 태그 입력 : 질문과 관련된 키워드를 입력합니다. 질문 활용과 공유를 위해서는 가급적 관련된 키워드를 정확하게 입력하는 것이 좋습니다. 키워드는 최대 10개까지 입력할 수 있으며, 키워드 사이에는 콤마(,)를 넣어서 구분합니다. 처음에 입력한 키워드는 후에도 언제든지 변경할 수 있습니다.

4. 답변하기

질문을 조회하면서 본인이 답변할 수 있는 내용이라면, 우측 하단의 '답변' 버튼을 클릭하여 답변 화면으로 이동한 후 답변을 입력하면 됩니다. 답변 요청을 받은 전문가가 아니더라도 누구나 답변을 제시할 수 있습니다.

질문 조회

질문 답변

답변 화면에서 질문에 대한 답변을 작성할 수 있습니다. 질문에 대한 답변도 내장된 웹편집기를 통해서 다양한 포맷으로 편집할 수 있습니다. 답변과 관련해서 참조할 파일이 있다면 질문 등록 때와 마찬가지로 파일로 첨부할 수 있습니다. 참고로, 답변에 대한 태그는 디펄트로 질문에 등록된 태그와 동일한 값이 세팅됩니다. 답변자가 변경하고 싶으면 변경할 수 있습니다.

5. 채택 및 활용하기

질문자는 제시된 답변 중에서 가장 도움이 된 답변을 채택할 수 있습니다. 질문자에 의해 채택된 답변은 가장 질문에 적합한 문제 해결 방법이라고 전제하고 다른 사용자들도 이를 참조할 수 있도록 '채택'된 답변으로 표시하고 있습니다. 여러 답변 중에서 가장 도움이 되는 답변을 '채택'하는 것은 질문자에게만 주어지는 기능입니다.

질문자에 의한 답변 채택

제시된 답변 중에서 질문자는 가장 마음에 드는 답변 조회 화면으로 이동한 후에 조회 화면 우측 하단의 '답변채택' 버튼을 클릭하여 가장 적합한 답변을 선택합니다. 질문자에 의한 답변채택이 이루어지면 해당 질문은 문제 해결이 완료된 건으로 간주됩니다.

채택 답변 조회

채택 답변은 채택되지 못한 답변들과 달리 'Choice'라는 마크가 부여됩니다. 여러 답변들 중에서 질문자에게 가장 도움이 되는 질문이라는 표시로 다른 사용자들에게도 어떤 답변이 가장 도움이 되는 답변이었는지 공유하기 위한 장치입니다. 질문자는 비록 채택한 답변이라도 '답변채택해제' 버튼을 클릭해서 언제든지 채택을 회수할 수 있습니다.

우수답변 추천하기

질문자 외의 사용자들도 답변 중에 추천할 만한 답변이 있다면 '추천' 버튼을 클릭해서, 해당 답변이 타 답변에 비해서 우수한 답변이라는 의사표시를 할 수 있습니다. 일정 수 이상의 '추천'을 받게 되면, 해당 답변은 비록 질문자에 의해 채택된 답변은 아니지만, 다른 많은 사용자들에 의해 우수한 답변으로 추천 받은 우수 답변(Best)으로 으로 표시됩니다. 관리자가 정한 우수답변 관리 정책에 의해 우수 답변이 되기 위한 기준 추천수가 정해집니다.

우수 답변이 되기 위찬 추천수를 충족하면 아래 그림과 같이 'Best' 마크가 표시됩니다. 아래 그림의 경우에는 질문자에 의해 채택도 되었고 질문자 외 다른 사용자들로 부터도 추천을 많이 받아서 우수 답변으로 선정된 경우입니다.
