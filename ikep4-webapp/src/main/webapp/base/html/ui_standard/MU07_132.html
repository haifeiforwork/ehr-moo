동맹이란?

닫기
  • ※ 다른 Collaboration과 동맹을 맺으면 동맹Collaboration에서 공개한 특정 게시판의 글을 조회할 수 있습니다.
  • ※ 동맹 Collaboration에 대한 게시판 공개여부 설정은 게시판 관리에서 할 수 있습니다.

A Collaboration

01. 동맹요청
동맹을 원하는 Collaboration을 지정하여 요청사유와 함께 동맹을 요청합니다.
  • 사용권한 : 시샵, 운영진
04. 동맹에서 공유된 게시판 조회
동맹관계인 Collaboration에서 공유한
게시판을 조회합니다
  • 사용권한 : 모든 사용자

B Collaboration

02. 동맹 승인
동맹 요청에 대하여 승인 또는 거부를
결정합니다.
승인한 경우 공유할 게시판을 지정합니다.
  • 사용권한 : 시샵
03. 동맹에게 게시판 공유
게시판 관리 메뉴에서 동맹관계인 Collaboration에 공유할 게시판을
지정합니다.(권한 설정)
  • 사용권한 : 시샵, 운영진
