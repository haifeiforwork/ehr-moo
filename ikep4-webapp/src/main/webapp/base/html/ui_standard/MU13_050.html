소방무선용 케이블 자료
홍길동 과장 2011.02.11
게이츠가 제품의 개선에 열정을 쏟는 데 비해 잡스는 완벽한 제품을 만들어내는 데 집념을 불태운다. 완성을 목표로 한 잡스의 열정은 대단하지만, 그가 정말 대단한 것은 완벽을 추구하다 실패한 뒤에 그 실패로부터 금방 다시 일어서는 집념이다.
