Overall Sociality Index

구분 Influence Index (0 < = x < = 100) Fellowship Index (0 < = x < = 100)
가중치 비율 70% 30%
  • Influence와 Fellowship의 값에 가중치를 부여하여 보정한 뒤 합산한 값을 사용합니다. 가중치는 백분율로 부여되며 각 가중치의 합은 100이 되도록 하여야 합니다.

Overall Sociality Index =

(Influence Index × 0.7) + (Fellowship Index × 0.3)

Influence Index

※ 최근 1개월에서 12개월 사이의 값으로 합니다.  최근 개월
항목 Follower 마이크로블로깅 Post수 마이크로블로깅 RePost수 Blog 방문수 Full Marks
보정값 2 1 5 1 200

Influence Value =

(Follower×2) + (트윗수×1) + (Retweet수×5) + (Profile 방문수×1);

Influence Index =

Influence Value >= 200 / 100 : (100×Influence Value / 200);

Fellowship Index

※ 최근 1개월에서 12개월 사이의 값으로 합니다.  최근 개월
항목 Follower Following Both Following 가입 Microblog Group수 방명록 방명록글수 Full Marks
보정값 2 1 10 10 1 200

Fellowship Value =

(Follower×2) + (Following×1) + (Mutual Following×10) + (가입 Microblog Group수 ×20) + (방명록글수×1);

Fellowship Index =

Fellowship Value >= 200 / 100 : (100×Fellowship Value / 200);
