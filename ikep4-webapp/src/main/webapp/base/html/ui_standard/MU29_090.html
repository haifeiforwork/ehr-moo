  • 구간점수계산
  • 저장
  • 심볼이미지(플레쉬 파일)은 10단계로 구성
  • PVI 각 단계별 구간적용값을 %단위로 입력가능(입력된 적용값은 총합이 100%가 되어야 하며 양의 정수값만 허용)
  • 구간점수계산 버튼을 클릭하면 각 단계별 구간적용값에 따라 PVI값을 10단계로 구분하여 점수구간을 산정함
    (예 : PVI 최대값 3586점 * 0.1 = 359점(반올림), 3586점 * 0.2 = 7171단계 : 0~359점, 2단계 : 360~1077, … 2500~3586)
  • 저장 버튼을 클릭하면 선택한 구간적용값과 구간점수를 적용함
