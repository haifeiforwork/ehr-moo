Social Distance 설정

Overall Social Network Distance

구분 Direct Network Distance (0 < = x < =1) Communication Network Distance (0 < = x < =1) Fellowship Network Distance (0 < = x < =1) Expertise Network Distance (0 < = x < =1)
가중치 비율 25% 25% 25% 25%
  • 각 4개의 Distance 값의 산술평균을 Overall Social Distance로 사용합니다. 다만, 각 Distance별로 가중치를 부여할 수 있으며, 가중치는 백분율로 부여되며 각 가중치의 합은 100이 되도록 하여야 합니다.

Overall Social Network Distance =

(Direct Network Distance × 0.25) + (Communication Network Distance × 0.25) + (Fellowship Network Distance × 0.25) + (Expertise Network Distance × 0.25)

Direct Network Distance

항목 팀원 0 or 1 팀장 0 or 1 Favorite 0 or 1 내가 Following 0 or 1 나의 Follower 0 or 1 Both Following 0 or 1
보정값 1 1 1 1 1 1
※ 보정값은 0 ~ 1000 사이의 값으로 합니다.

Direct Network Distance =

(같은팀원/1:0) + (팀장/2:0) + (Favorite/6:0) + (내가 Following/4:0) + (나의 Follower/2:0) + (Both/6:0) / ∑보정값;

Communication Network Distance

※ 최근 1개월에서 12개월 사이의 값으로 합니다.  최근 개월
항목 메일발송수 (0 < = x < =1) 쪽지발송수 (0 < = x < =1) SMS발송수 (0 < = x < =1) 방명록 글수 (0 < = x < =1)
최대값 100 10 10 5
보정값 4 10 10 5
※ 보정값은 0 ~ 1000 사이의 값으로 합니다.

Communication Network Distance =

(메일발송수>=100/100: 메일발송수/100)×4 + (쪽지발송수>=10/10: 쪽지발송수/10)×1 + (SMS발송수>=10/10: SMS발송수/10)×1 + (방명록글수>=5/5: 방명록글수/5)×4 / ∑보정값;
※ 메일발송수 외 각 항목값은 각각의 최대값을 기준으로 0 <= (항목값) <=1 로 정규화함

Fellowship Network Distance

※ 최근 1개월에서 12개월 사이의 값으로 합니다.  최근 개월
항목 나의 Follower 0 or 1 내가 Following 0 or 1 Both Following 0 or 1 나의 Comment수 0<=x<=1 나의 Mention수 0<=x<=1 상대방의 Comment수 0<=x<=1 상대방의 Comment수 0<=x<=1 공통 참여 Microblog Group개수 0<=x<=1
최대값 100 100 50 50 5
보정값 2 4 10 20 20 20 20 20
※ 보정값은 0 ~ 1000 사이의 값으로 합니다.

Fellowship Network Distance =

(나의 Follower/2:0) + (내가 Following/4:0) + (Both Following/10:0) + (나의Comment수 >=100?100: 나의Comment수/100)×20 + (나의Mention수 >=100/100: 나의Mention수/100)×20 + (상대방의 Comment수 >=50/50: 상대방의 Comment수/50)×20 + (상대방의 Mention수 >=50/50 : 상대방의 Mention 수/50)×20 + (공통Microblog Group수>=5/5:공통Microblog Group수 /5)×20 / ∑보정값;

Expertise Network Distance

항목 일치전문태그수 0<=x<=1 일치관심태그수 0<=x<=1 등록글 태그 일치 0<=x<=1 공통으로 답변한 질문수 0<=x<=1
최대값 3 4 5 3
보정값 10 2 3 20
※ 보정값은 0 ~ 1000 사이의 값으로 합니다.

Expertise Network Distance =

(일치전문태그수 >=3/3: 일치전문태그수/3)×10 + (일치관심태그수 >=4/4: 일치관심태그수/4)×2 + (등록글태그일치수 >=5/5 : 등록글 태그 일치수/5)×3 + (등록글태그일치수 >=3/3 : 등록글 태그 일치수/3)×20 / ∑보정값;
※ 실제 Distance 계산은 각 항목의 값은 전체 최대값을 기준으로 0과 1사이의 값으로 변환 후에 관리자가 입력한 보정값을 가하여 합산 후에 다시 각 항목별 보정값의 총합을 기준으로 0과 1사이의 값으로 변환한 값을 Distance로 합니다.
이 방법은 관리자가 중요하게 생각하는 항목에 대하여 유연하게 가중치를 부여할 수 있도록 합니다
