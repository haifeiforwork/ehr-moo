Method & Properties

  • url : 동적으로 가져올 경로
  • ref : 연관되는 select 엘리먼트, name이나 id 사용 - id의 경우 "#"을 붙혀 사용함.

* 해당 경로로 요청시 ref에 지정한 엘리먼트의 값을 파라미터로 전송하며 파라미터 이름은 ref에 지정한 이름이 된다(id의 경우 "#"은 배제됨)
