슬라이더의 value 값이 설정한 단위에 따라 증가/축소하는 예제. default 증가값은 1이다.

슬라이더의 range 옵션값을 true로 설정한 경우 두개의 값 사이를 설정할 수 있는 슬라이더를 생성한다.
두개의 값 사이는 다른 배경색으로 채워져 값을 선택하게 한다.
단, 두개의 선택 버튼이 겹치는 경우 왼쪽 선택 버튼이 우선이 된다.

슬라이더의 min 옵션을 이용하여 최소 시작값을 설정할 수 있으며.
특정값을 표시한 후 disabled:true 옵션을 통해 고정할 수 있다.

슬라이더의 방향을 orientation:"vertical" 옵션을 통해 세로로 설정할 수 있다.
슬라이더의 높이는 CSS의 높이 조절로 설정할 수 있다.

슬라이더의 방향을 orientation:"vertical" 옵션을 통해 세로로 설정할 수 있다.
슬라이더의 range 옵션값을 true로 설정한 경우 두개의 값 사이를 설정할 수 있는 슬라이더를 생성한다.

Overview

  • ui.handle : DOMElement - 현재 설정된 슬라이더 객체
  • ui.value : Integer - 현재 설정된 슬라이더 value값

Option

    • disabled
      • 슬라이더를 초기화 할 때 (true) 또는 (false) 값으로 설정되며, 슬라이더 값 변경기능을 제어한다.

      • Code examples
        							$( ".selector" ).slider({ disabled: true });
