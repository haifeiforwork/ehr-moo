슬라이더의 value 값이 설정한 단위에 따라 증가/축소하는 예제. default 증가값은 1이다.

슬라이더의 range 옵션값을 true로 설정한 경우 두개의 값 사이를 설정할 수 있는 슬라이더를 생성한다.
두개의 값 사이는 다른 배경색으로 채워져 값을 선택하게 한다.
단, 두개의 선택 버튼이 겹치는 경우 왼쪽 선택 버튼이 우선이 된다.

특정값을 표시한 후 disabled:true 옵션을 통해 고정할 수 있다.
