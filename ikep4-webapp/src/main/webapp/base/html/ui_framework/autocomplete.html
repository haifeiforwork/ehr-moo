JavaScript
	$("#tags").autocomplete({
		source : "ajax_result/autocomplete_items.json",	// source에서의 result 중 첫번째 아이템이 객체형으로 쓰여지면 이후 문자형 아이템으로 혼영해서 사용하면 오류발생
		change : function(event) {
			iKEP.debug($(this).val());
		},
		select : function(event, item) {
			this.blur();	// 아이템 선택 후 포커스 이동시 change이벤트 발생하므로...
			iKEP.debug(item);//label, value
		}
	});

Method & Properties

  • .autocomplete(options)
    • options
      • source : array / url(ajax request url)
        * url 지정시 해당 url로 요청시 "term" 파라미터 이름으로 입력한 값 전송
      • change : callback_function(event) - 지정된 input의 값이 변경되었을때 발생하는 이벤트
      • select : callback_function(event, item) - 제시된 아이템을 선택했을때 발생하는 이벤트
        item : {label:"", value:""} - source가 label, value쌍의 오프젝트 리스트일 경우 해당하는 값으로 설정되며, 그렇지 않으면 같은 값으로 설정됨.
