K-WAH3.0

K-WAH는 웹 사이트 접근성을 증진시키기 위해 한국정보화진흥원에서 제작한 프로그램이다.
이 프로그램은 자동 평가도구로 모든 웹 접근성 지침 준수 여부를 판단하지는 못한다.
실제로는 체크리스트를 보며 직접 소스를 일일이 검수해야 한다.

검수 순서는 다음과 같이 하는 것이 효율적이다.

  • 1. 웹표준 Validator Pass (Error 외에 Warning도 웹 접근성과 관계된 부분이 있으므로 반드시 모두 pass하도록 한다.)
  • 2. 웹접근성 K-WAH 프로그램 오류 정정 (오류는 반드시 고치되 경고는 특별한 경우가 아니라면 참고만 한다.)
  • 3. 웹접근성 수동 평가 (UI표준과 함께 검사)
